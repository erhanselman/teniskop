# Wimbledon Tekler Şampiyonları

Wimbledon Championships, ilk kez 1877 yılında düzenlenen dünyanın en eski tenis turnuvasıdır. Kadınlar tekler kategorisi ise 1884 yılında başlamıştır.

Aşağıda Wimbledon tekler şampiyonlarının kronolojik tarihçesi yer almaktadır. Kadınlar tekler kategorisi 1884 yılında başladığı için ilk yıllarda yalnızca erkekler şampiyonu bulunmaktadır. 

- 1915–1918 yılları arasında I. Dünya Savaşı nedeniyle,
- 1940–1945 yılları arasında II. Dünya Savaşı nedeniyle,
- 2020 yılında  COVID-19 pandemisi nedeniyle **turnuvalar düzenlenmemiştir**.
